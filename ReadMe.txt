Pour pouvoir lancer l'application, il est nécessaire d'avoir la librairie JFoenix Java 8 : https://github.com/jfoenixadmin/JFoenix
Il est nécessaire d'importer ce jar dans le projet sous IntelliJ : Project Structure > Librairies > Add JFoenix.jar

Pour lancer le jeu version console lancer le main DomiNations Console
Pour lancer le jeu version applicative lancer le main DomiNations

Nous avons aussi utilisé JUnit pour les cas de tests au début
